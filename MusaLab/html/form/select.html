A Cosa serve il tag Select

I suoi Utilizzi

Il tag Select è un elemento del form, serve per far selezionare delle
opzioni predefinite all'utente.
Questo elemento ha due atributi fondamentali oltre quelli standard

  • size (Definisce quanti elementi vengono mostrati senza scroll)
  • multiple (consente di selezionare più elementi)
Questo elemento ha l'unico figlio option che serve per specificare ogni singola
opzione che l'utente sceglie. il suo attributo Value specifica quale valore inviare al server

Al tag select è necessario assegnare un nome con l'attributo "name", sia per il recupero dei
dati lato server che per la verifica delle selezione lato client. L'attributo "id", invece,
è obbligatorio ai fini del rispetto delle specifiche dell'XHTML.
